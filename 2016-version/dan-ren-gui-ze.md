# 單人規則

## 起始設定

使用兩人遊戲時的起始設定，來對抗 Lacerda：

1. Lacerda 不用玩家版塊，設置他的錢(只會賺不會花)、酒桶、專家板塊
   1. 四種專家板塊各一個
   2. 一開始沒有錢
   3. 全部的玩家酒桶
2. 洗混兩堆牌：九張行動牌，九張號碼牌
3. 起始果園，抽一張號碼牌，把最上面一張果園棄掉。但還是要放一個名望方塊。
4. 起始巨頭板塊，抽一張號碼牌左上至右下 1-6。
5. 放兩個酒桶到出口區兩人區域的左上兩格
6. Lacerda 分數從 19 分開始

## 回合流程

按照回合順序，玩家和 Lacerda 都各自做完兩動才推進回合指示物一格，然後換另一方執行兩動。

### Lacerda 回合

當 Lacerda 行動時，每個行動從行動牌抽一直執行後棄掉。行動牌的棄牌會在每次評鑑大會後洗回牌庫。

### Lacerda 移動

1. Lacerda 絕對不會付錢，如果他進入玩家的地區，玩家是跟銀行拿錢。
2. Lacerda 移動時會在原地留下一塊錢做記號，如果玩家移動進入該地區要付一塊錢給銀行。然後再把地區上的一塊錢也還給銀行。玩家如果付不出錢就不能前往對應的行動格。

### Lacerda 的行動卡：

#### 果園 Vineyards:

抽兩張號碼牌，棄掉對應的果園最上面一張。各放一個名譽方塊。

#### 地窖 Cellars:

抽兩張號碼牌，放名譽方塊到對應地區。然後棄掉兩張地窖。（就算只剩一張地窖也是抽兩張號碼。）

#### 釀酒廠 Wineries:

抽兩張號碼牌，放名譽方塊到對應地區。然後棄掉兩張釀酒廠。（就算只剩一張地窖也是抽兩張號碼。）

#### 品酒專家 Wine experts:

每個種類各拿一張放到 Lacerda 這邊。

果農/釀酒師 Enologists/Farmers

丟掉果農和釀酒師各兩個。

#### 出口 Export:

1. Lacerda 拿走玩家持有的地區中有最多名譽方塊的，兩個。（平手就抽號碼卡直到先抽到的先選）
2. 放一個 Lacerda 到出口區最高價值的空格。如果有複選，選最左邊的。
3. Lacerda 立刻得到對應的品質加上用掉的名譽方塊的分數。
4. 如果還有酒桶就重複 1-3 一次。

#### 販售 Sales:

1. Lacerda 拿走玩家持有的地區中有最多名譽方塊的，兩個。（平手就抽號碼卡直到先抽到的先選）
2. 抽號碼牌直到大於 2
3. Lacerda 賣出的酒的價值 = 號碼卡 ＋ 名譽方塊數量
4. Lacerda 酒的種類由當年度收穫板塊中巨頭 Ａ 喜愛的種類決定
5. 放置酒桶到對應的顏色和價值的地方：
   1. 如果已經被擺放了，放動同價值的另一個種類。
   2. 如果也被擺放了，降低一點價值，放到原先的種類，依此類推。
   3. 如果低分都沒有空位，重複 3-5 直到有。
6. Lacerda 得到對應的錢。
7. 如果還有酒桶或空位，就再執行一次 1\~6

#### 跳過/提早曝光 Pass / Press Release:

每次執行這個行動，Lacerda 都會決定先動。

1. 增加 Lacerda 對應收穫板塊上，巨頭 B 期望的酒的品質的評鑑分數。
2. Lacerda 拿走玩家持有的地區中有最多名譽方塊的，兩個。（平手就抽號碼卡直到先抽到的先選）
3. Lacerda 會優先選擇 3 點評鑑分數的展位，如果被佔領了就選 3 塊錢的展位。
4. 如果 Lacerda 有符合的專家板塊，他會毫不猶豫地用掉來增加評鑑分數。

#### 維持階段 Maintenance：

Lacerda 拿回最低價值的銷售區酒桶。如果平手，會拿當年度巨頭 A 喜歡的種類。

#### 評鑑大會 Fair:

如果 Lacerda 還沒提交酒會先進行一次。

評鑑會排名計分跟兩人遊戲一樣，得第一名和第三名的分數。而且額外得到一個獎杯（用未使用的玩家圓形指示物替代）

輪到 Lacerda 挑選巨頭板塊的時候，抽選數字卡直到抽出數字  1\~6 ，如果已經被玩家選走了就繼續抽，直到分數卡抽完，後始為 PASS。Lacerda 拿到的巨頭板塊會直接棄掉，拿到終局獎勵板塊會留著並放個空酒桶上去。如果沒有空的酒桶，他就不會繼續玩了。每一個終局獎勵最後都會給 Lacerda 10 分。

### 最後結算：

1. 如果分數比 Lacerda 多：得四點榮譽
2. 如果錢比 Lacerda 多：得兩點榮譽
3. 每一場評鑑會贏過 Lacerda：得一點榮譽
4. 每填滿一個莊園：得一點榮譽
5. 專家板塊得分比 Lacerda 多：得一點榮譽
6. 如果從巨頭拿回自己所有酒桶：得一點榮譽

5+: Wine Snob

10+: Wine Producer
